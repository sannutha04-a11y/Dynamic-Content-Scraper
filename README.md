# Dynamic Content Scraper

## Overview
A simple Python project to scrape content from dynamic websites using Selenium and BeautifulSoup. It includes a GUI built with Tkinter.

## Features
- Opens a dynamic website in a browser
- Waits for full content to load
- Extracts and displays text content
- Saves content to a .txt file

## Technologies Used
- Python
- Selenium
- BeautifulSoup
- Tkinter
